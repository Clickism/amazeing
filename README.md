# Amazeing

Amazeing is an educational project consisting of maze-solving challenges that are solved
using an assembly-like language called *Amazeing Assembly*.

## Tech Stack

Built using **React + TypeScript**.

## Tasks

Tasks are handled under `src/features/precourse/tasks`.

- Under this folder, in the `days.json5` file, it is defined which days have which tasks.
  (Corresponding to the exercise sheets)
- Tasks are defined in subfolders that are referenced by the `days.json5` file.
